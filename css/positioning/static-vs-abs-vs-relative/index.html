Static vs Absolute

Static

static

This box has position: static (default value). The element will be positioned according to the normal document flow, meaning it will occupy space in the layout without any adjustments from top, left, or other offset properties

Absolute with ancestor position: non relative

absolute

The position: absolute element will position itself relative to the nearest positioned ancestor (non-static). If no such ancestor exists, it defaults to being positioned relative to the initial containing block (ICB), typically the root element.
